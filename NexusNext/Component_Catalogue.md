# Component Catalogue

## Core Layout Components
- **`AppLayout`**: Primary structural wrapper handling sidebars and responsive content spacing.
- **`NxTopBar`**: Omni header with global search command bars, job monitoring, and theme switches.
- **`GlobalJobMonitor`**: Persistent foreground modal sliding in from the right to monitor active system jobs.
- **`NxCommandBar`**: Spotlight-style command pallet.

## Data Display & Containers
- **`NxGlassCard`**: Fundamental container utilizing translucent backgrounds and blurred backgrounds. 
- **`NxContactCard3D`**: Immersive contact showcase card with 3-dimensional on-hover lighting.
- **`NxEmptyState`**: Scalable fallback UI for empty database screens.
- **`NxTable` / `NxTableRow`**: Structured list designs for heavy data (Contacts grid to list view).

## Analytical & Visuospatial Controls
- **`NxActivityHeatmap`**: Grid-based layout resembling GitHub commits for cognitive trace tracking.
- **`NxMemoryMiniGraph`**: Horizontal spectrum percentage graph for index taxonomies.
- **`NxTagCloud`**: Dynamic font-size tag agglomeration component.
- **`NxRelationTimeline`**: Interactive timeline connecting specific intelligence events.

## Form Elements
- **`NxActionButton`**: Versatile interactive button with loading, icons, and variants.
- **`NxInput`, `NxSelect`, `NxSwitch`**: Standard styled form inputs.
- **`NxChatInput`, `NxChatBubble`**: Specific UI modules for conversation windows.
